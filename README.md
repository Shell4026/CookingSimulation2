# VR 요리 시뮬레이션 프로젝트
 
요리시뮬레이션 폴더의 Scene에서 작업합니다.

추가 할 스크립트는 요리시뮬레이션 폴더의 Scripts폴더 내에 잘 정리해서 추가합니다.

# 커밋 규칙
추가한 내용이 있으면 
feat: 추가한 내용

수정한 내용이 있으면
fix: 수정한 내용

사소한 수정이 있으면
chore: 수정한 내용

커밋 되돌리기, 이전 내용으로 바뀌기 등
revert: 내용

기능 추가를 할 땐 Branch를 하나 만들어서 커밋하기
